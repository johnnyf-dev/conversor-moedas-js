💱 Currency Converter
A project built with pure JavaScript that converts between Brazilian Real (BRL), US Dollar (USD), and Euro (EUR), with a dynamic and responsive interface.

✨ Features
🏳️ Automatic update of flags and currency names.

🔄 Conversion between Real, Dollar, and Euro.

🎨 Display formatted with Intl.NumberFormat.

🖱️ Conversion when clicking the button.

🔄 Conversion when changing selects.

⌨️ Conversion in real time while typing.

🌐 Initial conversion when loading the page.

🛠️ Technologies used
⚡ JavaScript (ES6+)

🎨 HTML5

🎨 CSS3

🖼️ DOM manipulation

📌 Execution Flow
🧩 moedas() → Updates flags, names, and resets values.

🔢 conversaoDeValores() → Performs conversion calculations.

🎯 formatacao() → Applies formatters and displays values.
